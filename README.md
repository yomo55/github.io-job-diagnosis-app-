# お仕事診断アプリ

あなたらしい働き方を見つけるための診断アプリです。

## 特徴

- 5つの質問に答えるだけで、あなたに合ったお仕事タイプがわかります
- 女性向けのやさしいデザイン
- スマホ・PC対応のレスポンシブデザイン
- 診断結果に応じたPDF案内機能付き

## 使い方

[診断アプリを開く](https://あなたのGitHubユーザー名.github.io/job-diagnosis-app/)

## カスタマイズ

`index.html` の以下のセクションを編集することで、質問内容や診断結果をカスタマイズできます：

- 質問内容: `questions` 配列
- 診断結果: `resultTypes` オブジェクト
- PDFのURL: 各診断結果の `pdfUrl` プロパティ
